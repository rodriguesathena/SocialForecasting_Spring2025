Question 4

#non-stationary
ggAcf(data_ts) +  ggtitle("Original ACF Plot")

Original ACF plot: There is high autocorrelation at multiple lags which can be seen in spikes at seasonal intervals, every 12 months. This shows a strong seasonality that is expected with monthly electricity generation data. If there were a quick drop in ACF to zero the data would be stationary. However, the slow decay in autocorrelation indicated non-stationarity.

Addressing ACF: To properly model this data differencing is needed to remove trends and stabilize data. This could be done through seasonal differencing (lag =12) that would prepare the data for an ARIMA or seasonal ARIMA model which could account for trend and seasonal autocorrelation.

#seasonal differencing
differencing <- diff(data_ts, lag = 12)
ggAcf(differencing) +
  ggtitle("ACF Plot after Seasonal Differencing")

Differencing ACF Plot: After applying seasonal differencing as mentioned above, the ACF plot shows a significant reduction in autocorrelation with a decrease of most lags and the elimination of seasonal spikes. The removal of the seasonal component has moved the series closer to stationarity.

Using checkresiduals shows how well the ARIMA model fits the electricity data by checking the leftover patterns through the Ljung-Box Test and various plots.

Ljung-Box Test: This test checks if residuals are random with a p-value greater than 0.05 being significant of randomness achieved. Because the p-value is small, randomness of residuals is not achieved. This suggests that the model may have missed some patterns in the data with some time structure at seasonal lags still present.

Residuals Over Time: The top plot shows the actual data versus what the model predicted. For a good model, the residuals should shift around zero with no distinct pattern. In this plot the residuals are random and evenly spread which is good however the larger spikes showcases some remnants of the seasonal spike.

ACF Plot: The bottom left checks if there is any pattern of seasonality or time remaining. Most lags should stay within the blue lines to show there is no strong autocorrelation left. Majority of the bars in this plot stay within the lines showing the model is able to handle the time-relations well, however there is still excess lag seen at varied periods.

Residuals: The bottom right shows the distribution of residuals. This should show as a bell curve, highlighted in red, that is centered at zero representing an overall normal distribution. The plot largely follows this curve although there is a skew left showing some error has occurred.
